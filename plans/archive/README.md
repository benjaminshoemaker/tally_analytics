# Archived Plans

This directory contains historical planning artifacts. They are preserved for context, but they are not the active product plan.

Archived material includes:
- `greenfield/` — original SaaS analytics platform plan.
- `features/github_oauth/` — GitHub OAuth feature plan.
- `features/improved_metrics/` — V2 metrics and conversion-tracking plan.
- `features/stripe_integration/` — billing feature plan.

Agents should not implement from these documents unless the human explicitly names one of them as the source to revive. For new planning, create a fresh current plan or discovery document outside this archive.
